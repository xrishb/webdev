Document Object Model

Introduction to DOM

The DOM is an interface that allows a programming language to manipulate the content, structure, and style of a webpage. JavaScript is the client-side scripting language that connects the DOM to an internet browser. The DOM is language-agnostic, and the browser is just a program that interprets HTML and CSS and renders it.

In addition to parsing HTML and CSS, the browser creates a representation of a document, known as the Document Object Model. This model allows JavaScript to access the content and elements of a website document as objects. The DOM is modified by client-side JavaScript, and the browser automatically fixes errors in the source code.

DOM Tree

The DOM is often referred to as the DOM tree and consists of tree-like objects called nodes. 

        
            <a href="dom.html">DOM</a>

Here we have an anchor element, which is a link to dom.html:

  • `a` is the tag.
  • `href` is the attribute.
  • `dom.html` is the attribute value.
  • `DOM` is the text inside the anchor tag.
