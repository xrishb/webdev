CSS box model

Introduction to Box Model

The CSS box model is essentially a box that wraps around HTML elements, and it consists of:

  • margin: area around the border (outer area of an element), does not have background color.
  • border: that goes around padding and the content, effected by the bg of the box.
  • padding: area around the content (inner area of an element)
  • actual content: the content
  • element dimensions: Width and height

Inline vs Block display

All elements are assigned a display property of either inline or block.

  • inline: horizontally placed elements such as links, media, etc.
  • block: elements stack on top of the other as blocks.

Tips

  • A div is just a block element that we define ourselves.
  • A span is an inline element.
  • A class is a named collection of CSS properties used for span,div or any other element.

  • CSS box model only applies to the block elements and not to inline elements.
  • if two block elements are next to each other and their margin is greater than 0, the larger of two margins will be applied.
